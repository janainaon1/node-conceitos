## Running the Samples From the Command Line

1. Clone this repository:
```bash 
git clone https://github.com/janainaon1/node-conceitos.git
```

2. Entry the folder:
```bash 
cd node-conceitos
```

3. Install dependencies:
```bash 
npm i
```

4. Run the application:
```bash 
node server.js  
```
